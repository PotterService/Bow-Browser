# Bow Browser

Welcome to Bow Browser!

Bow Browser is a custom desktop browser designed to provide a personalized browsing experience while serving as the gateway to the growing BowVerse ecosystem. Built with future expansion in mind, Bow Browser aims to bring together custom tools, extensions, vaults, and web projects into a single easy-to-use platform.

## Features

* Modern browser interface
* Custom Bow Home Page
* Quick access to BowVerse projects
* Integrated access to Bow Extension Depot
* Support for custom Bow extensions
* Future-ready design for additional features
* Standalone desktop application

## Planned Features

* Custom themes
* Bookmark manager
* Extension manager
* Built-in Bow Extension Depot
* MiniNet integration
* Creator Vault integration
* Grade Vault integration
* Closet Vault integration
* Media Vault integration
* Enhanced privacy options
* Custom new tab page
* User profiles

## Installation

### Development Version

1. Install Node.js
2. Download or clone this repository
3. Open a terminal in the project folder
4. Install dependencies

```bash
npm install
```

5. Launch Bow Browser

```bash
npm start
```

## Building an Executable

To build a Windows installer:

```bash
npm install electron-builder --save-dev
npx electron-builder --win
```

The generated installer will be located in the `dist` folder.
